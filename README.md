## 🌌 Parallax Website

A modern **Parallax Scrolling Website** built using **HTML, CSS, and JavaScript**.  
This project demonstrates how parallax effects create an immersive scrolling experience where background elements move at different speeds than foreground elements.

## 🚀 Live Demo
🔗 https://parallextarun.netlify.app/

## 📌 Features
- Smooth Parallax Scrolling Effect
- Responsive Design
- Clean UI
- Lightweight and Fast
- Cross Browser Support

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Netlify (Deployment)
